## üíª Challenge Requirements

[**Front End Mentor**](https://www.frontendmentor.io/) challenges are evaluated by adherence to the provided design files. The objective is to produce a solution as close to the example as possible with the given resources.

**Recipe Page** tests basic HTML5 and CSS3 skills to provide a page displaying an Omelette Recipe. Responsive layouts for Mobile, Tablet, and Desktop are created to provide a linear experience across devices.

With all Front End Mentor challenges, a Figma design file has been provided to reference various design decisions (fonts, colors, margins/paddings) and constraints.
